# energy-distribution-neutrinos

# ice cube neutrino energy analysis 

## overview
this project analyzez neutrino energy and direction using IceCube public data (2008-2018)

## Goals and Methods
- distribution of energy analysis
- analysis of neutrino directionality 
- track vs cascade event analysis

## Structure
- jupyter notebooks/: analysis and graphical imagery 
- src/: reusable functions
- data/: dataset storage
- figures/: plots

## How to run
pip install -r requirements.txt
jupyter notebook
